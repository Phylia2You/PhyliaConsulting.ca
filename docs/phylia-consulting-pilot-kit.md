# Phylia Consulting Pilot Kit

## Positioning

**PhyliA Consulting helps small businesses organize customer inquiries, reply templates, follow-ups, and practical AI-friendly admin workflows without turning it into a giant software project.**

## First service

### Customer Response Mini-Cleanup

A small, practical setup for businesses that answer the same questions repeatedly, lose track of inquiries, or forget follow-ups.

**Deliverables**

1. FAQ / common question list
2. Polished reply templates
3. Simple inquiry tracker in Airtable
4. Follow-up templates
5. AI reply helper prompt
6. One-page usage guide

**Boundary for early pilots**

This pilot does **not** include complex automation, custom software, medical/legal/financial advice, full CRM migration, guaranteed sales, or handling sensitive client records.

---

## Client intake questionnaire

Use these questions in an Airtable form, email, or a simple document.

1. Business name
2. Contact name
3. Email and phone
4. What type of business do you run?
5. Where do customers contact you? Phone, email, website form, Facebook, Instagram, text, booking app, other?
6. What are the 5–10 questions customers ask most often?
7. Which questions are annoying or time-consuming to answer repeatedly?
8. Do inquiries ever get missed or forgotten? If yes, where does that usually happen?
9. Do you currently follow up when someone asks a question but does not book/buy?
10. What tools do you currently use? Website, email, Google Workspace, Microsoft 365, Airtable, CRM, booking app, phone/SMS, other?
11. What tone should replies have? Warm, professional, brief, detailed, friendly, formal?
12. Are there topics that should never be answered automatically?
13. What would make this project feel useful to you?
14. May I use an anonymous before/after summary as a case study?

## Internal workflow

1. Add the inquiry to Airtable table **Inquiries**.
2. Send the **New inquiry reply** template.
3. Send the intake questionnaire.
4. Review answers and identify the smallest useful improvement.
5. Create a short mini-audit with current inquiry sources, repeated questions, follow-up gaps, and recommended starter system.
6. If it is a pilot, confirm scope in writing.
7. Build the mini-cleanup: FAQ list, reply templates, inquiry tracker, follow-up templates, AI helper prompt, and one-page guide.
8. Deliver with a short explanation.
9. Ask for feedback and testimonial.
10. Add lessons learned to **Client Problems** and improve templates.

---

## Mini-audit checklist

- [ ] Is the business clear about where inquiries arrive?
- [ ] Are common questions documented anywhere?
- [ ] Are replies consistent?
- [ ] Is there one place to track inquiry status?
- [ ] Is there a next follow-up date?
- [ ] Are there sensitive topics that require human review?
- [ ] Can this be improved with templates before automation?
- [ ] Is Airtable/spreadsheet enough for version 1?
- [ ] Is there one clear next step for the owner?

## AI reply helper prompt

Use this prompt to draft replies from approved FAQ/template content. Never use it for medical, legal, financial, or sensitive advice.

```text
You are helping draft a customer service reply for [Business Name].
Tone: warm, clear, professional, and concise.
Use only the approved information below. Do not invent policies, prices, availability, guarantees, medical/legal/financial advice, or anything not included.

Customer message:
[Paste customer message]

Approved business information / FAQ:
[Paste relevant FAQ/template notes]

Draft a reply that:
1. Answers the question clearly.
2. Mentions any next step.
3. Says when a human should confirm details if needed.
4. Keeps the message friendly and not too long.
```
